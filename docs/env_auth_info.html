env_auth_info – Gather information about CDP environment authentication details¶

  • Synopsis

  • Requirements

  • Parameters

  • Examples

  • Return Values

  • Status

Synopsis¶

  • Gather information about CDP environment authentication details, notably the FreeIPA root certificate and user keytabs.

  • The module supports check_mode.

Requirements¶

The below requirements are needed on the host that executes this module.

  • cdpy

Parameters¶

Parameter

Choices/Defaults

Comments

name


list

A target list of environments or a single environment string. If no environments are specified, all environments are targeted.


Aliases: environment

root_certificate


bool

A flag indicating whether to retrieve the given environment’s FreeIPA root certificate.


Aliases: root_ca, cert

keytab


bool

A flag to retrieve the keytabs for the given environment or environments, governed by the value of user. If no environments are declared, all environments will be queried.


Aliases: keytabs, user_keytabs

user


list

A list of user IDs or a single user ID for retrieving the keytabs from the specified environment(s). If no user ID is declared, the current CDP user will be used.


Aliases: users

verify_tls


bool

Verify the TLS certificates for the CDP endpoint.


Aliases: tls

debug


bool

Capture the CDP SDK debug log.


Aliases: debug_endpoints

profile


str

If provided, the CDP SDK will use this value as its profile.

Examples¶

# Note: These examples do not set authentication details.

# Retrieve only the root certificate for a single environment
- cloudera.cloud.env_auth_info:
    name: the-environment
    root_certificate: yes
    keytab: no

# Retrieve the root certificate for multiple environments
- cloudera.cloud.env_auth_info:
    name:
      - one-environment
      - two-environment
    root_certificate: yes
    keytab: no

# Retrieve the keytab details for the current CDP user for selected environments
- cloudera.cloud.env_auth_info:
    name:
      - one-environment
      - two-environment
    keytab: yes
    root_certificate: no

# Retrieve the keytab details for the specified users for selected environments
- cloudera.cloud.env_auth_info:
    name:
      - one-environment
      - two-environment
    user:
      - UserA
      - UserB
    keytab: yes
    root_certificate: no

Key

Returned

Description

dict

Returns a dictionary of the environment authentication details.

certificates

_environment name_


str

always

The FreeIPA root certificate for the environment

keytabs


dict

when supported

A dictionary of the keytabs for each specified environment by user.

_workload username_


dict

always

The user’s workload username.

_environment name_


str

always

The keytab for the environment. The keytab is encoded in base64.

sdk_out


str

when supported

Returns the captured CDP SDK log.

sdk_out_lines


list

when supported

Returns a list of each line of the captured CDP SDK log.
