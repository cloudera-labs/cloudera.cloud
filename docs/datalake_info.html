datalake_info – Gather information about CDP Datalakes¶

  • Synopsis

  • Requirements

  • Parameters

  • Examples

  • Return Values

  • Status

Synopsis¶

  • Gather information about CDP Datalakes

Requirements¶

The below requirements are needed on the host that executes this module.

  • cdpy

Parameters¶

Parameter

Choices/Defaults

Comments

name


str

If a name is given, that Datalake will be described. If no name is given, all Datalakes will be listed and (optionally) constrained by the environment parameter.


Aliases: datalake

environment


str

The name of the Environment in which to find and describe the Datalake.


Aliases: env

verify_tls


bool

Verify the TLS certificates for the CDP endpoint.


Aliases: tls

debug


bool

Capture the CDP SDK debug log.


Aliases: debug_endpoints

profile


str

If provided, the CDP SDK will use this value as its profile.

Examples¶

# Note: These examples do not set authentication details.

# List basic information about all Datalakes
- cloudera.cloud.datalake_info:

# Gather detailed information about a named Datalake
- cloudera.cloud.datalake_info:
    name: example-datalake

# Gather detailed information about the Datalake in an Environment
- cloudera.cloud.datalake_info:
    environment: example-env
